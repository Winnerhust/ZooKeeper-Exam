ZooKeeper-Exam
==============

ZooKeeper练习代码


* ClusterMonitor  ： 集群监控和Master选举
* Config : 远程配置管理，根据[inifile2](https://github.com/Winnerhust/inifile2)修改而成，主要修改了open函数，其他不变
* Notify : 分布式通知/协调
* NameService: 命名服务
* Lock: 分布式锁
* Queue: 分布式队列
* mymetaq: 负载均衡，实现了生产者负载均衡，消费者负载均衡尚未完成
